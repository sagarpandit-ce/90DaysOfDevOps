# Linux Commands Cheat Sheet

## File System Commands

* `pwd` - Show current directory
* `ls` - List files and folders
* `cd` - Change directory
* `mkdir` - Create a new folder
* `rmdir` - Remove an empty folder
* `touch` - Create a new file
* `cp` - Copy files or folders
* `mv` - Move or rename files
* `rm` - Delete files
* `cat` - View file contents

## Process Management Commands

* `ps` - Show running processes
* `top` - Monitor system processes
* `kill` - Stop a process using PID
* `pkill` - Stop a process by name
* `jobs` - Show background jobs
* `bg` - Run a job in background
* `fg` - Bring a job to foreground

## System Information Commands

* `whoami` - Show current user
* `uname -a` - Show system information
* `df -h` - Show disk usage
* `free -h` - Show memory usage
* `uptime` - Show system uptime

## Networking Commands

* `ping` - Check network connectivity
* `ip addr` - Show IP address information
* `curl` - Transfer data from a URL
* `dig` - DNS lookup
* `ss` - Show network connections

## Log and Service Commands

* `systemctl status` - Check service status
* `journalctl` - View system logs
* `tail -f logfile` - Monitor log file in real time

Today I learned commonly used Linux commands for file management, process management, networking, and troubleshooting. These commands are useful for daily Linux and DevOps tasks.
