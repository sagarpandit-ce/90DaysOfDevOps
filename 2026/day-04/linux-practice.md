# Linux Practice - Day 04

Today I practiced some Linux commands.

## Commands Used

### ps

Shows running processes.

### top

Shows CPU, memory and running processes.

### pgrep bash

Shows the PID of the bash process.

### uname -a

Shows system information.

### cat /etc/os-release

Shows operating system details.

### ls /var/log

Lists available log files.

### tail -n 20 /var/log/*.log

Shows recent log entries.

## What I Learned

Today I learned how to check running processes, view system information and read log files. These commands are useful for basic Linux troubleshooting.
