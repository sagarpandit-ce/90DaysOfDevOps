# My 90 Days DevOps Learning Plan

## Current Level
I’m a Computer Engineering student and currently exploring DevOps from scratch.

## What I Understand About DevOps & Cloud
For me, DevOps is about learning how real applications are deployed, managed, automated, and maintained using modern tools and cloud platforms.

## Why I Started Learning DevOps
I started learning DevOps because I want to build practical skills, work on real projects, and become more confident with tools that are actually used in the industry.

## My Goals for the Next 90 Days
1. Get comfortable with Linux and networking basics
2. Learn Docker, Kubernetes, and CI/CD step by step
3. Stay consistent and build a strong GitHub profile through daily practice

## Core Skills I Want to Improve
- Linux and troubleshooting
- Docker & Kubernetes
- Git and CI/CD workflows

## Final Goal
By the end of these 90 days, I want to feel confident working with DevOps tools and be ready to build real-world projects and prepare for opportunities in DevOps and Cloud.
